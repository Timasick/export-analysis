# export-analysis
Анализ международной торговли (1988–2021).

# Export / Import Data Analysis

## Описание проекта
Данный репозиторий содержит лабораторные работы по анализу международной торговли и построению моделей машинного обучения.

В рамках проекта выполняется:
- анализ данных международной торговли
- очистка и подготовка данных
- создание новых признаков
- построение первой модели машинного обучения

Анализ выполняется в среде **Jupyter Notebook** с использованием Python.

---

# Используемые технологии

- Python
- pandas
- numpy
- matplotlib

- Jupyter Notebook

# Лабораторная работа №1  
## Анализ данных международной торговли

В работе выполнено:

- загрузка датасета
- первичный анализ данных
- анализ импорта по странам
- определение топ-10 стран по объему импорта
- анализ динамики мирового импорта

Основные библиотеки:
- pandas
- matplotlib

---

# Лабораторная работа №2  
## Очистка данных и подготовка признаков

Выполнены следующие этапы:

- анализ категориальных признаков
- кодирование категориальных переменных
- масштабирование числовых признаков
- создание новых признаков
- удаление нерелевантных признаков
- подготовка финального датасета для модели

Использованные методы:

- One-Hot Encoding
- Label Encoding
- StandardScaler

---

# Лабораторная работа №3  
## Построение модели машинного обучения

В работе реализовано:

- выбор целевой переменной
- разделение данных на обучающую и тестовую выборку
- обучение модели линейной регрессии
- оценка качества модели
- анализ результатов

Метрики качества:

- MAE
- RMSE
- R²
